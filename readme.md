# Primix 🚀

Первый в мире бекенд-язык с соединением по ключам и T8T шифрованием.

## Что такое Primix

Primix — это простой язык для создания серверов. Серверы находят друг друга по ключам, а не по IP-адресам. Весь трафик шифруется через T8T.

## Возможности ✅

- Связь серверов по ключам 🔑
- T8T шифрование (Mixed, Japanese, English, Random) 🔐
- Режим призрака 👻
- SOS — авто-переброс задач при падении 🆘
- Кроссплатформенность 💻📱
- Простой синтаксис 📝
- Изоляция маршрутов 🛡️
- JSON + MessagePack 📦
- SQLite 🗄️
- Кеш, очереди, балансировка ⚡
- Физический доступ только с места 🏠

## Быстрый старт ⚡

Установи Python 3.8+
Скачай файлы
Запусти:

python primix.py examples/server.pmx

Открой браузер: localhost:5847/hello

## Пример кода 💻

start run/server

go request /hello then
    respond "Hello, World!"

go request /api/data then
    respond json {"status": "ok"}
